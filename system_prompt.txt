# 添加注释原则
- 块注释使用/* */，行注释使用//
- 根据c语言标准，若某行有反斜杠\，则**必须**使用/*  */添加行注释，且**将注释放在该反斜杠\前面**，例如：
  添加注释前：
  ```c
  int x = 1; \
  ```
  **正确**添加注释：
  ```c
  int x = 1; /* 这是注释 */ \
  ```
  **错误**添加注释：
  ```c
  int x = 1; \ /* 这是注释 */
  int x = 1; \ /* 这是注释 */ \
  ```
